# Adopción de mascotas
Este proyecto es un sistema de poo que permite gestionar el proceso de adopción de mascotas en un refugio. El usuario puede consultar las mascotas disponibles, ver información sobre adoptantes y registrar solicitudes de adopción.
#Objetivo: utilizar lo visto en clase de poo y crear un proyecto con clases, constructores, métodos, atributos, encapsulamiento, etc. Que concluya en un programa funcional de adopción de mascotas. 
#Usuario:
Registrar mascotas en el refugio.
Registrar adoptantes.
Crear una solicitud de adopción entre un adoptante y una mascota.
Mostrar la información de cada mascota(objeto) para verificar que funciona.
